- Surface cards: rgba(255,255,255,0.04)
- Card borders: 1px solid rgba(255,255,255,0.08)
- Primary text: #f0ede8 (warm off-white)
- Muted text: rgba(240,237,232,0.5)
- Accent: #f5a623 (electric amber) — used sparingly 
  for maximum impact on headlines, numbers, CTAs, 
  and key moments only
- Light section background: #f7f4ef (warm off-white) 
  used for the process section only as a contrast break

Typography:
- Display font: "Fraunces" from Google Fonts
  Used for all headlines and large numbers
  Italic variant for emphasis words in headlines
- Body font: "DM Sans" from Google Fonts
  Used for all body copy, UI, labels, nav
- Import both via Google Fonts link tag

Type scale:
- Hero H1: clamp(3.2rem, 7vw, 6rem), weight 700, 
  line-height 1.0, letter-spacing -0.03em
- Section titles: clamp(2rem, 4vw, 3.2rem), weight 600
- Body: 16px, line-height 1.7, weight 400
- Labels: 11px, uppercase, letter-spacing 0.1em, muted
- Card body: 14px, line-height 1.65

Layout principles:
- Asymmetric and editorial — not a SaaS template
- Large typographic watermark section numbers 
  (01, 02, 03...) in the background of each section 
  at 3% opacity in accent color
- Generous whitespace — sections breathe
- Strong visual hierarchy — the headline always dominates

Buttons:
- Primary: solid accent (#f5a623) background, 
  #0f0f0f text, sharp corners (border-radius 4px), 
  padding 14px 28px, font-weight 500, font-size 14px
- Secondary: transparent background, 1px solid accent, 
  accent text, same padding and radius
- Hover: primary darkens 10%, secondary fills lightly

Cards:
- Background: rgba(255,255,255,0.04)
- Border: 1px solid rgba(255,255,255,0.08)
- Border-radius: 12px
- Hover: border brightens to rgba(245,166,35,0.3), 
  transition 0.2s

Animations:
- Page load: hero headline words animate in 
  sequentially, each word fades up with 80ms delay
- Scroll: IntersectionObserver fadeUp on all sections
  opacity 0 to 1, translateY 20px to 0, 
  transition 0.6s ease, staggered 80ms per child
- Marquee trust bar: seamless CSS infinite scroll
- Accent pulse: small dot before eyebrow badge 
  pulses with CSS animation

COPY RULES — ENFORCE ON EVERY LINE OF COPY
═══════════════════════════════════════════════════════════════

- Direct and peer-level — never salesy or deferential
- Short punchy paragraphs — no walls of text
- No passive voice anywhere on the page
- No hyphens and no em dashes anywhere
- No AI buzzwords: leverage, utilize, synergy, 
  cutting-edge, revolutionary, game-changing, 
  unlock, empower, transform
- "We" and "you" throughout — never third person
- Every sentence should sound like it was written 
  by someone who has actually shipped these systems
- Numbers are always specific — never vague ranges 
  when an exact number exists
- Scarcity is real — "3 spots this month" not 
  "limited availability"
